# Daily Dose Coffee, clean Vite landing page

This is a stripped-down Vite + React landing page version of the Daily Dose project.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy on Vercel

- Framework preset: Vite
- Build command: npm run build
- Output directory: dist
- Root directory: leave blank
